盘点说明

交易类型：充值、账期还款、订单结款、订单退款


支付方式：支付宝、微信、货到付款、账期、线下、余额



1.盘点流程：新建-->选择仓库-->选择分类-->下载导出模板-->上传数据表-->校验-->提交(生成盘点单)


2.库存盘点计算公式：


  盈亏数=盘点数量-实际库存

  更新后实际库存=盘点数量

  更新后可用库存=盘点数量-（实际库存-可用库存）


3.数据表上传校验条件：


   if(更新后可用库存>=0)

   {

      if(盘点盈亏数>0的所有商品)

        {

             生成一张入库单

         }

      else if（盘点盈亏数<0的所有商品）

        {

             生成一张出库单

         }

       else (盘点盈亏数=0的所有商品)

        {

              不生出入库单

        }


    }else

   {

       禁止上传，并提示不满足条件的商品

   }
